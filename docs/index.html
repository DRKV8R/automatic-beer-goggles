Getting Started

1

Installation

Clone the repository and install dependencies:

git clone https://github.com/DRKV8R/automatic-beer-goggles.git
cd automatic-beer-goggles
pip install -r requirements.txt
2

Master Audio

Master your audio file with professional quality:

python -m automatic_beer_goggles.cli master-audio \
  input.wav mastered.wav --target-lufs -14.0
3

Create Video

Generate platform-optimized videos:

python -m automatic_beer_goggles.cli create-video \
  image.png audio.wav output.mp4 \
  --platform youtube --aspect-ratio 16:9
4

Batch Process

Process multiple files for all platforms:

python -m automatic_beer_goggles.cli batch-process \
  audio_dir/ image.png output/ \
  --platforms youtube instagram tiktok
